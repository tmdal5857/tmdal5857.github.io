Project description

'유럽여행, 한 페이지에서 해결'

유럽은 나라도 많고 정보도 굉장히 방대합니다.
친구들과 유럽여행을 다녀오며 겪었던 경험을 바탕으로 여행을 준비하는 사용자들에게 다른 포털의 수많은 검색 없이 우리의 웹페이지만으로도 웬만한 일정계획을 마무리 하고, 여러 알짜배기 정보들을 손쉽게 볼 수 있는 페이지를 제작하였습니다.
또한 후기,마켓,동행 게시판을 통하여 여러 사용자와 소통할 수 있습니다.
로그인을 통해 직접 그 나라의 여행을 내 여행에 추가하여 여행 계획을 할 수 있는 플랫폼입니다.
현재 전세계가 COVID-19로 인해 여행은 물론, 바깥을 자유롭게 돌아다니지 못하고 있습니다. 현 시국을 고려하여 집에서 웹페이지 하나만으로 여행을 누리는 느낌을 줄 수 있습니다.

Role

독일, 스위스, 이탈리아 페이지 UI 구성, 관련 데이터베이스 설계, 동행 게시판 기능, 맡은 기능에 대한 Controller, Action, DAO, DTO 작성

Function:

게시판 기능, 로그인/회원가입, 내 여행 추가, 여행에 필요한 정보제공, 댓글 기능
